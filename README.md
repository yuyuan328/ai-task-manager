## 功能简介

AI 任务调度官，不是简单列待办，而是根据 **DDL 优先级、历史完成速度、科目难度**，帮你智能安排今天最该先做什么。录入任务后持续跟踪完成情况，AI 动态调整后续安排。

### 核心功能

| 功能 | 说明 |
|------|------|
| 📝 任务录入 | 添加任务（名称、DDL、预计耗时、科目、难度） |
| 🤖 AI 智能排序 | AI 综合多维度分析，给出今日最优执行顺序 |
| ✅ 完成追踪 | 记录实际耗时，AI 学习你的完成速度 |
| 🚨 DDL 预警 | AI 识别紧急任务并给出时间分配建议 |
| 📊 周报总结 | 每周分析完成情况，识别拖延模式 |

### AI 核心价值

- **多维度综合分析**：DDL + 难度 + 耗时 + 历史模式 → 最优排序（纯规则做不到）
- **个性化学习**：根据你的历史完成速度动态调整预估
- **拖延模式识别**：发现你最容易拖延的任务类型并预警
- **教练式建议**：不只是排序，还告诉你"怎么做"和"什么时候做"

## 快速开始

### 1. 安装依赖

```bash
pip install openai
```

### 2. 配置 API Key

程序已内置 DeepSeek API Key，可直接运行。如需修改，编辑 `skill/scripts/task_manager.py` 中的 `CONFIG` 字典。

### 3. 运行

```bash
cd skill
python scripts/task_manager.py
```

### 4. 使用预置测试数据

```bash
python scripts/task_manager.py --task-file ../data/tasks.json
```
